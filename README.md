## Why Will-Be ?

### 요즘 화상 면접 안보는 곳 있나요? WillBe와 함께 준비해볼까요 👨‍💻

면접을 연습한 적은 있지만, 본인의 모습을 직접 녹화해 보신 적 있으신가요?<br/>
